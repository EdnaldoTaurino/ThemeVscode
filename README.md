## Edy Theme

Edy Theme é um tema escuro moderno para o Visual Studio Code.

## Instalação

Você pode instalar o Edy Theme diretamente do Marketplace do Visual Studio Code.

## Características

- Escuro e moderno
- Suporte para várias linguagens
